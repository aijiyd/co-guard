# 论文项目概览

## 基本信息

- 论文类型：本科毕业论文
- 学科领域：网络空间安全、大语言模型安全、知识图谱推理
- 暂定题目：面向多轮子任务拆解攻击的知识图谱推理防御方法
- 输出格式：Markdown
- 引用格式：GB/T 7714 风格
- 写作语言：中文

## 研究背景

大语言模型逐渐被用于智能体系统、自动化任务处理和多 Agent 协作场景。攻击者可以将原本有害的目标拆分为多个表面中立的子任务，并在多轮交互中逐步获取完成攻击所需的信息。传统单轮输入检测难以识别这种跨轮上下文中的意图累积。

## 研究目标

本文设计并实现 Co-Guard 防御系统，将用户多轮查询转化为上下文知识图谱，通过语义抽取、图谱构建、relation-first 多跳搜索和上下文组装评分器判断当前查询是否与历史查询共同形成潜在恶意意图链。

## 研究方法

- EDC 风格语义抽取：Extract、Define、Canonicalize
- 上下文知识图谱：查询节点、实体节点、关系边、session 隔离
- 图推理判定：relation-first 多跳搜索、风险链路评分、LLM judge / rules / hybrid
- 实验评估：基于 `data/advbench_decomposed.jsonl` 的顺序注入测试

## 实验依据

- 场景一全量实验：拦截率 89.42%，攻击穿透率 10.58%
- 场景二全量实验：拦截率 99.23%，攻击穿透率 0.77%
- 场景三全量实验：拦截率 99.42%，良性误拒率 72.97%
